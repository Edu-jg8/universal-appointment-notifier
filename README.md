# üîî Universal Appointment Notifier (v1.0)

A high-reliability Python automation tool designed for small businesses to eliminate no-shows. This system sends personalized email reminders directly from your Gmail account to your clients, ensuring a professional and consistent communication flow.



## üöÄ Key Features

* **Global Date Intelligence**: Robust parsing engine that automatically recognizes International (ISO(YYYY/MM/DD)) and European (DD/MM/YYYY) date formats.
* **Smart CSV Ingestion**: Equipped with an automatic delimiter detector (supports `,` and `;`) and a "noise filter" that skips empty rows or phantom columns often generated by Excel or Google Sheets.
* **Security First**: Credential management via environment variables (`.env`) and secure encryption through SMTP_SSL.
* **Human-Like Delivery**: Integrated "Smart Delay" between emails to protect your Gmail account's reputation and ensure high deliverability.
* **Professional Logging**: Generates a detailed `activity.log` for real-time monitoring and troubleshooting.


3.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

## üìã Usage Guide

1.  **Prepare your Data**: Use Google Sheets or Excel to manage your schedule. Ensure you have these headers: `DATE`, `TIME`, `CLIENT_NAME`, `EMAIL`, `SERVICE`, `LOCATION`.
2.  **Export & Save**: Save your file as `appointments.csv` inside the `data/` folder.
3.  **Run the App**: Execute via terminal or use the provided `run_app.bat` file:
    ```bash
    python main.py
    ```
4.  **Monitor**: Check `logs/activity.log` to verify that all reminders were sent successfully.
